Curso Node.js Fundamentos y Web Apps

¿Acabas de empezar a aprender Node.js y deseas una base sólida?
En el curso fundamentos se te proporcionarán todas las herramientas necesarias para dominar este framework mediante el aprendizaje de su core y arquitectura.

En el curso Web Apps aprenderás a construir aplicaciones web, conectadas a bases de datos y con capacidad de responder en tiempo real.
¿Algún requisito? Conocimiento básico en JS y ganas de aprender!
